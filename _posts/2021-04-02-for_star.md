#### 자바 이중 for문으로 별찍기
오랜만에 자바 복습할 겸 별찍기 하고 있는 나.. 학원 초반에는 하루 종일 걸렸는데 지금와서 하니 그래도 전보다 오래 걸리지는 않네😂😂

#### 5*5 별찍기

- 이고 찍어봅시다!<br>
<span>\*\*\*\*\*</span><br>
<span>\*\*\*\*\*</span><br>
<span>\*\*\*\*\*</span><br>
<span>\*\*\*\*\*</span><br>
<span>\*\*\*\*\*</span><br>


```java

 //5*5 별찍기
  for(int i=0; i<5; i++) {
    for(int j=0; j<5; j++) {
      System.out.print("*");
    }
    System.out.println();
  }

```
